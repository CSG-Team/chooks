# useTimeout

一个可以处理 setTimeout 计时器函数的 Hook。


## 使用
```tsx
import React, { useState } from 'react';
import { useTimeout } from 'cchooks';

const Timeout = () => {
  const [state, setState] = useState(1);
  useTimeout(() => {
    setState(state + 1);
  }, 3000);

  return <div>{state}</div>;
};

```
